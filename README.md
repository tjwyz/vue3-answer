# vue3-answer

项目运行（nodejs 6.0+）
# 克隆到本地
git clone https://github.com/Oudanlu/vue3-answer

# 进入文件夹
cd vue3-answer

# 安装依赖
npm install 或 yarn(推荐)

# 开启本地服务器（localhost:8088）
npm run dev

# 发布环境
npm run build
